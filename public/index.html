🔐 Linkify – Key Vault Demo

A public/private RSA key-gated JSON data store

Step 1 – Public Story GET /api/data

Any visitor can fetch this endpoint. It returns the "cover story" from db.json. 

Click "Fetch story" to load…

Step 2 – Server Public Key GET /api/public-key

Retrieves the server's RSA public key. Copy it into Step 3 to prove you hold the matching private key. 

Click "Fetch server public key" to load…

Step 3 – Submit Public Key POST /api/data

Paste an RSA public key (PEM format) below. If it matches the server's key, you receive the secret payload encrypted with your key. 

Response will appear here…
